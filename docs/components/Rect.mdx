

# Rect矩形

## 示例
```tsx preview
import {useEffect,useState,useRef} from 'react';
import {Stage,Rect} from 'graphify-chart';
function App() {
  const chartRef = useRef(null);
  useEffect(() => {
    const stage = new Stage(chartRef.current);
    const rect = new Rect(stage);
    stage.appendChild(rect)
  },[])
  return (<div id="rect" ref={chartRef} style={{height:200}} ></div>)
}
export default App;
```
## 属性支持

| 属性 | 说明  | 类型 | 默认值 |
| :-- | :---: | :--- | :---: |
| x | x坐标 | ?number | 0 |
| y | y坐标 | ?number | 0 |
| width| 宽 | ?number | 100 |
| height | 高 | ?number | 100 |
| fillStyle | 填充色 | ?string | "blue" | 
| borderColor | 边框颜色 | ?string | "rgba(0,0,0,0)" |
| borderWidth | 边框宽度 | ?number | 0 |
| radius | 图形圆角 | ?number | 0 |
| zIndex | 层级 | ?number | 0 |
