

# Sector扇形

## 示例
```tsx preview
import {useEffect,useRef} from 'react';
import {Stage,Sector} from 'graphify-chart';
function App() {
  const chartRef = useRef(null);
  useEffect(() => {
    const stage = new Stage(chartRef.current);
    const sector = new Sector(stage,{x:100,y:100});
    stage.appendChild(sector)
  },[])
  return <div ref={chartRef} style={{height:200}} ></div>
}
export default App;
```
## 属性设置
当我们想修改矩形的某一项属性时，可以通过节点内置的`setAttr`方法修改节点的属性，示例如下:
```tsx preview
import {useEffect,useState,useRef} from 'react';
import {Stage,Sector} from 'graphify-chart';
function App() {
  const chartRef = useRef(null);
  const sectorRef = useRef(null)
  useEffect(() => {
    const stage = new Stage(chartRef.current);
    const sector = new Sector(stage,{x:100,y:100});
    sectorRef.current = sector
    stage.appendChild(sector)
    return () => {
      stage.unMount()
    }
  },[])
  return (<div>
    <div ref={chartRef} style={{height:200}} ></div>
    <input type="color" onChange={evt => sectorRef.current.setAttr({fillStyle:evt.target.value})} />
  </div>)
}
export default App;
```
## 坐标
同样的我们可以通过设置`x`,`y`来修改节点的位置，和圆形一样，扇形也是xy坐标是圆心的位置。示例如下：
```tsx preview
import {useEffect,useRef} from 'react';
import {Stage,Sector} from 'graphify-chart';
function App() {
  const chartRef = useRef(null);
  useEffect(() => {
    const stage = new Stage(chartRef.current);
    const sector = new Sector(stage,{x:150,y:100});
    stage.appendChild(sector)
  },[])
  return <div ref={chartRef} style={{height:200}} ></div>
}
export default App;
```
## 尺寸
可以通过设置`radius`来设置圆半径来展示不同尺寸的圆。示例如下：
```tsx preview
import {useEffect,useRef} from 'react';
import {Stage,Sector} from 'graphify-chart';
function App() {
  const chartRef = useRef(null);
  useEffect(() => {
    const stage = new Stage(chartRef.current);
    const sector = new Sector(stage,{x:100,y:100,radius:100});
    stage.appendChild(sector)
  },[])
  return <div ref={chartRef} style={{height:200}} ></div>
}
export default App;
```
## 样式
可以通过设置`borderColor`、`borderWidth`和`radius`来实现更丰富的样式实现。示例如下：
```tsx preview
import {useEffect,useRef} from 'react';
import {Stage,Sector} from 'graphify-chart';
function App() {
  const chartRef = useRef(null);
  useEffect(() => {
    const stage = new Stage(chartRef.current);
    const sector = new Sector(stage,{x:100,y:100,fillStyle:'yellow',borderColor:'#13C2C2',borderWidth:2});
    stage.appendChild(sector)
  },[])
  return <div ref={chartRef} style={{height:200}} ></div>
}
export default App;
```
## 透明度
可以通过设置`opcity`修改图形的透明度。下面我们来对比一下两个圆这只不同透明度的区别：
```tsx preview
import {useEffect,useRef} from 'react';
import {Stage,Sector} from 'graphify-chart';
function App() {
  const chartRef = useRef(null);
  useEffect(() => {
    const stage = new Stage(chartRef.current);
    const sector = new Sector(stage,{x:100,y:100,});
    const sector2 = new Sector(stage,{x:150,y:100,opcity:0.5});
    stage.appendChild(sector,sector2)
  },[])
  return <div ref={chartRef} style={{height:200}} ></div>
}
export default App;
```
## 属性支持

| 属性 | 说明  | 类型 | 默认值 |
| :-- | :---: | :--- | :---: |
| x | x坐标 | ?number | 0 |
| y | y坐标 | ?number | 0 |
| radius | 半径 | ?number | 50 |
| fillStyle | 填充色 | ?string | "blue" | 
| borderColor | 边框颜色 | ?string | "rgba(0,0,0,0)" |
| startAngle | 起始角度 | ?number | 0 |
| endAngle | 结束角度 | ?number | Math.PI * 1.5 |
| zIndex | 层级 | ?number | 0 |
| opcity | 透明度 | ?number | 1 |
